# Desafio VIAMAKER

Projeto consiste em fazer uma aplicação web consistindo no front-end e back-end, tecnologias utilizadas React, Typescript, Styled-Components, Node.JS e Sequelize.

## Instalação

Para poder rodar o projeto será necessário abrir as pastas front e back separadamentes, atentar-se ao readme de cada um para rodar o projeto.
